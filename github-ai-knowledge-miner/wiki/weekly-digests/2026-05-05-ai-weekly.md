## Summary / 摘要

This week’s useful signal is not “more agents,” but stronger control over execution, state, memory, and traces. / 本周真正有用的信号不是“更多 agent”，而是对执行、状态、内存和 trace 的控制更强。

## Knowledge Points / 知识点

- [Constrained Agent Execution / 受约束 Agent 执行](../patterns/constrained-agent-execution.md): use explicit constraints and human decisions to keep long agent workflows on track. / 用显式约束和人工决策让长流程 agent 不跑偏。
- [Single Source State Ownership / 单一状态所有权](../patterns/single-source-state-ownership.md): centralize schema or prompt ownership to reduce drift across layers. / 把 schema 或 prompt 的所有权集中，减少跨层漂移。
- [KV Cache Memory Efficiency / KV Cache 内存效率](../patterns/kv-cache-memory-efficiency.md): reduce inference memory pressure through low-precision cache or modality-aware compression. / 通过低精度 cache 或模态感知压缩降低推理内存压力。
- [Observability Context Propagation / 可观测性上下文传播](../patterns/observability-context-propagation.md): attach spans to active context so downstream tools preserve trace hierarchy. / 把 span 附着到活跃上下文，让下游工具保住 trace 层级。
